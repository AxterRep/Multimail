# Multimail
Generador de correos con alias de Gmail para crear cuentas ilimitadas desde un solo email.

# Requisitos
- Linux (Kali, Ubuntu, etc.)
- Python 3
- Bash

# Instalar
```bash
git clone https://github.com/AxterRep/Multimail.git
cd Multimail
chmod +x install.sh
./install.sh
```

# Instalar en Termux
```bash
apt-get install python
git clone https://github.com/AxterRep/Multimail.git
cd Multimail
chmod +x generar.sh
./generar.sh
```

# Desinstalar
```bash
rm -rf /home/axter/Multimail && rm ~/Escritorio/Multimail.desktop && rm ~/.local/share/applications/Multimail.desktop
```
# Notas
- Funciona únicamente con Gmail
- El truco de los puntos no funciona en Hotmail, Outlook o Yahoo
- El símbolo + puede ser bloqueado por algunos sitios web
